# Chat Simulator

React & Redux를 이용하여 채팅 기록을 재생할 수 있도록 하는 과제입니다.

## Setup

Install dependencies

```sh
$ yarn install (or npm install)
```

## Development

```sh
$ yarn start (or npm start)
# visit http://localhost:3000
```

## Features

- 데이터베이스에서 채팅기록을 불러와 재생시킵니다.
- 마크다운과 링크는 슬랙과 같이 변환됩니다.
- 현재 접속자 목록을 볼 수 있습니다.

## Skills

- Redux
